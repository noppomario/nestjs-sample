# NestJS Sample

## 概要

[Nest](https://github.com/nestjs/nest)フレームワークを使ったサンプル

## 事前準備

- Node.js の導入
- SQLite3 の導入

## 導入

```bash
$ npm install
```

## 起動コマンド

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## テスト

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
